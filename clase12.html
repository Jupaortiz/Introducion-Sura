Posiciones con CSS

Para enstalbecer las posicion que los elementos de Html van a ocupar dentro del espacio de la estructura, se controlara la propiedad cde CSS position, esta nos permite controlar las distancias arriba, derecha, abajo e izquierda(top,roght, left, bottom) de un elemneto(contenedor), para es paropiedad exien 5 valores diferentes

  1. Static
  2. Relative
  3. Absolute
  4. Fixed
  5. Sticky

Posicion Static

Este es el valor por defecto de los elementos, posicionandolos con el flujo normal e la estructura, sin posibilidad de ajustar su posicion. Ej:

Esta div contiene la poscion static

Posicion Relativ

Con este tipo de posicion podemos mover el contenedor de su lado original, con respecto a los 4 lados, sin embargo esta posicion no afecta a otro elemnto, de tal forma que ningun contenido se ajuste para encajar en el espacio dejado por un elemnto que tenga esta posicion

Esta div contiene posicion relative

Posicion Absolute

El elemneto con este tipo de posicion se meuve aligual en sus cuatros lados, pero este cumple funciones caracteristicas distintas:

1. Cuando aplicmos esta propiedad cambiamos el flujo normal del contenido, aplicando los cuatro lados de orientacion.

2. Un vez cambiamos la posicion de un elemnto con esta propiedad, los demas elemntos de su alrededor ocuparan el espacio dejao por este.

3. Los elementos con este tipo de posicion pueden superponerse sobre los demas elementos.

4. Si esta propiedad se aplica a un elemento sin estar por dentro de otro con distinta posicion, este se movera y buscara al contenedor mas cerano o por defecto sin mover al inicio de la pantalla

Posicion Fixed

Un elemento con esta posicion se coloca en relacion a la ventana del navegador, lo que significa que siempre permanece en el mismo luegar, incluso si se desplaza la pagina, se utilizan los cuatro lados para determinar su posicion, este valor no deja el pacio en la pagina donde originalment esta ubicado

Posicion Sticky

Un elemnto con esta posicion se ubica en funcion del desplzacamiento del usuario dentro de la pagina,se posiciona en un lugar especifico mientras que se desplaza pero siempre vuelve a su lugar original
